# AadikaviAwakens 🌟

## Project Overview

**AadikaviAwakens** is an innovative digital platform that bridges the gap between classical Nepali literature and modern artificial intelligence. Named after Bhanubhakta Acharya, Nepal's revered "Aadikavi" (first poet), this project explores the intersection of traditional poetry, cultural heritage, and cutting-edge AI technology.

The project combines literary exploration, AI research, and educational outreach to create an immersive experience that celebrates Nepal's rich poetic tradition while embracing the possibilities of artificial intelligence.

🌐 **Live Website**: [https://upesh0070.github.io/AadikaviAwakens/](https://upesh0070.github.io/AadikaviAwakens/)

---

## 🎯 Mission Statement

Our mission is to:
- **Preserve and Celebrate**: Honor the literary legacy of Bhanubhakta Acharya and classical Nepali poetry
- **Innovate**: Explore the convergence of traditional literature with modern AI capabilities
- **Educate**: Provide accessible resources for learning about Nepal's literary heritage
- **Inspire**: Bridge cultural understanding through the universal language of poetry and technology
- **Connect**: Build a community of learners, poets, and AI enthusiasts who appreciate both tradition and innovation

## 💡 Inspiration

This project draws inspiration from:

### 🏛️ **Bhanubhakta Acharya (1814-1868)**
- Nepal's first poet to translate the Ramayana into Nepali
- Pioneer of accessible literature for the common people
- Symbol of cultural awakening and linguistic pride

### 🤖 **AI and Understanding**
- The quest to understand the "number 42" - a reference to Douglas Adams' "The Hitchhiker's Guide to the Galaxy" and the ultimate question of life, the universe, and everything

- The intersection of human creativity and machine intelligence

### 🌊 **The Alpha Rising**
- Represents the beginning of a new era in digital humanities
- The awakening of classical wisdom in the modern age
- The first wave of AI-enhanced cultural preservation

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS for customization
- Interest in literature, AI, or Nepali culture!

### Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/ripplewave2025/AadikaviAwakens.git
   cd AadikaviAwakens
   ```

2. **Local Development**
   ```bash
   # Open index.html in your browser
   open website/index.html
   
   # Or use a local server
   python -m http.server 8000
   # Then visit http://localhost:8000/website/
   ```

3. **GitHub Pages Deployment**
   - The project is automatically deployed via GitHub Pages
   - Visit the live site at the URL provided above
   - Changes to the `master` branch trigger automatic deployments

### Quick Start Guide

1. **Explore the Website**: Visit the live site to experience the interactive elements
2. **Browse the Documentation**: Check the `docs/` folder for detailed information
3. **Examine the Resources**: Explore `resources/bhanubhakta_lore/` for literary content
4. **View Images**: The `Images/` folder contains visual assets and artwork
5. **Contribute**: Follow our contribution guidelines to add your own enhancements
